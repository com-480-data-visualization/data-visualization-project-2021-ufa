arXiv dataset in graphs

This is a tool that allows you to visualize through interactive graphs the scientific publications in the arXiv dataset and it is not endorsed by arXiv.

i

The slider allows you to select a period of time in which papers were published on arXiv and this is reflected in the visualizations below.

Graph of paper categories

i

This is a weighted graph of all categories of papers in the arXiv database. The edges are proportional to the number of papers having the two categories at the same time. You can click and drag the nodes. When you click or hover a node you can see the adjacent categories and additional statistics below.

papers

Cloud of papers

i

Each point represents a paper which you can either click on to reveal its name and categories or search in the bar below to discover its position. The cloud was built using PCA on the title/abstract.
